 Conducted a comparative analysis of ChatGPT and Stack Overflow for security-related programming queries using PMD tools. Analyzed 953 code examples, revealing ChatGPT’s higher rate of security violations (70%) compared to Stack Overflow (18%). Stack Overflow
 responses were preferred in 76% of cases, highlighting AI limitations in security.
